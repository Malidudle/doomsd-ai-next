Command Bar + Toast Response

Full-width input bar pinned to bottom. AI responses appear as a brief toast overlay that auto-fades after 5s. Existing controls shift up slightly. Minimal screen real estate used.

Key Elements

  • Command bar — always visible at bottom, green border-top, terminal prompt feel
  • Toast response — appears above command bar on AI reply, fades after 5s. Shows AI text + action taken.
  • Existing controls — shift up by ~44px to sit above the command bar
  • No chat history — command bar is fire-and-forget. Previous commands don't persist on screen.
